# Jogo da Cobra (Snake Game)

Este é um jogo da cobra simples desenvolvido em HTML, CSS e JavaScript. O jogo consiste em controlar uma cobra em um campo delimitado, com o objetivo de comer alimentos para crescer e marcar pontos. No entanto, o jogador deve evitar colidir com as bordas do campo ou com o próprio corpo da cobra, o que resulta no fim do jogo.

# Funcionalidades

### Pontuação: 
- A pontuação é contabilizada conforme a cobra come os alimentos. Cada vez que a cobra come um alimento, a pontuação aumenta.
### Game Over: 
- O jogo termina quando a cobra colide com as bordas do campo ou com seu próprio corpo.
### Reinício: 
- Após o fim do jogo, o jogador pode reiniciar o jogo pressionando o botão "Jogar Novamente".

## Instruções de Uso
Abra o arquivo index.html em um navegador da web.
Use as teclas de seta do teclado para controlar a direção da cobra: cima, baixo, esquerda e direita.
Tente comer o máximo de alimentos possível sem colidir com as bordas do campo ou com o corpo da cobra.
Quando o jogo terminar, pressione o botão "Jogar Novamente" para reiniciar.
Estrutura de Arquivos
index.html: O arquivo HTML principal que contém a estrutura do jogo.
style.css: O arquivo CSS que define o estilo visual do jogo.
script.js: O arquivo JavaScript que contém a lógica do jogo.
Pré-requisitos
Não há nenhum requisito específico para executar este jogo além de um navegador da web moderno que suporte HTML5 e JavaScript.
